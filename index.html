3MF Print File Extractor Tool

Extract all print settings from a .3mf file into a searchable table, then copy it as Markdown.

Built for BambuStudio and OrcaSlicer, and compatible with many 3MF files that include project settings data Runs 100% locally in your browser — your file never leaves your device

What is the 3MF Print File Extractor Tool?

A free, browser-based utility for 3D printing users. Upload one .3mf file and extract its print settings into a clear table you can search and copy.

What data is extracted?

The tool reads Metadata/project_settings.config inside your 3MF file, parses the settings JSON, and flattens every value into key paths like filament_type[0] so you can inspect or share them quickly.

How do I share the data?

After extraction, click Copy as Markdown Table. Paste it into forums, Reddit, chat apps, or AI tools to get print troubleshooting help without sharing your 3D model file.

Privacy & Security

  • All processing happens locally in JavaScript
  • Your file is never uploaded to any server
  • No extracted data is transmitted anywhere
  • The only external resource is JSZip loaded with a security hash (SRI)
